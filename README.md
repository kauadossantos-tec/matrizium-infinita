# 🌌 Matrizium Infinita
> **Status:** Operacional | **Core:** v1.0.0 | **Estética:** Visceral / Tech-Horror

**Matrizium Infinita** não é apenas um cenário de ficção; é um **sistema operacional narrativo**. Um ecossistema compartilhado onde o rigor técnico do versionamento de dados (Git) se funde à densidade do terror cósmico e da fantasia sombria. 

Aqui, o *worldbuilding* é tratado como arquitetura de software: modular, escalável e imune à entropia narrativa.

---

## 👁️ O Conceito
A Matriz é uma estrutura de nodos interconectados. Cada conto, personagem ou local é um dado processado que deve respeitar os protocolos de realidade estabelecidos no **Core Canon**. Inspirado pela atmosfera de Junji Ito e a complexidade de Lovecraft, o projeto busca explorar a "corrupção da alma" através de uma linguagem técnica e descritiva.

---

## 🛠️ Arquitetura do Repositório

O projeto segue uma estrutura de diretórios padronizada para garantir a integridade referencial via [Obsidian](https://obsidian.md/):

```text
/matrizium-infinita
├── 📂 .protocols/         # Regras imutáveis e Leis da Realidade (Bíblia)
├── 📂 atlas/              # Nodos geográficos e cartografia digital
├── 📂 chronos/            # Linhas do tempo e logs históricos (Timeline)
├── 📂 entities/           # Personagens, facções e seres corrompidos
├── 📂 artifacts/          # Itens, tecnologias e sistemas de magia
└── 📂 src/                # Scripts Node.js para validação e automação
